Synapse

Conductor.Synapse — Type
struct Synapse <: Conductor.AbstractSynapse

A synapse (edge) between two neurons in a network.

  • source

    Presynaptic neuron/subcompartment.

  • target

    Postsynaptic neuron/subcompartment.

  • conductance

    Synaptic conductance model of the synapse.

  • reversal

    Synaptic equilibrium/reversal potential.

NeuronalNetworkSystem

Conductor.NeuronalNetworkSystem — Type
struct NeuronalNetworkSystem <: Conductor.AbstractNeuronalNetworkSystem

A network of neurons with synaptic connections.

  • iv::Symbolics.Num

    Independent variable. Defaults to time, $t$.

  • synapses::Vector{Conductor.AbstractSynapse}

    Vector of synapses (edges) between neurons in the network.

  • extensions::Vector{ModelingToolkit.AbstractTimeDependentSystem}

    Additional systems to extend dynamics. Extensions are composed with the parent system during conversion to ODESystem.

  • name::Symbol

  • eqs::Vector{Symbolics.Equation}

  • systems::Vector{ModelingToolkit.AbstractTimeDependentSystem}

  • observed::Vector{Symbolics.Equation}
